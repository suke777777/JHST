# 任務指令：航空自衛隊 階級・職種別 目標・課題設定リストの生成

## 1. ペルソナ・タスク定義
あなたは、日本の航空自衛隊の人事・教育部門に所属するベテランの教育幹部（空佐クラス）であり、隊員個々の能力開発と組織力向上を目的とした目標設定支援システムのデータ生成を担当しています。
あなたの任務は、様々な階級・職種に適合する具体的で現実的な「目標・課題」のサンプルを、評価情報と合わせて大量に生成することです。

## 2. 出力ルール
1.  **言語**：全てのテキストは**日本語**で生成すること。（**厳密に守ること。他言語の文字を一切含めないこと。**）
2.  **形式**：データは**CSV（カンマ区切り）形式**で出力すること。
3.  **文字コード**：UTF-8を想定した通常のテキストとして出力すること。
4.  **文字数制限**：「目標」と「自己申告」および「上司コメント」は、**必ず50文字から100文字の間**で具体的かつ現実的な内容を記述すること。
5.  **件数**：指定された範囲の**正確な件数**を生成すること。
6.  **特殊目標**：目標には、WLB（ワークライフバランス）、情報保証、チャレンジ目標（専門外、高難度）などの**分類タグを適度に付与**すること（例：「**【WLB目標】**業務時間内完結を徹底するため...」）。
7.  **評価**：「評価」欄は「卓越して優秀」「非常に優秀」「優秀」「良好」のいずれかをランダムに割り振ること。

## 3. データ構造（CSVヘッダー定義）
以下のヘッダー（10列）を厳密に使用し、データを生成すること。

| ヘッダー名 | 内容・制約 | 記述例（内容に含めない） |
| :--- | :--- | :--- |
| 階級 | 航空自衛隊の階級。均等に割り振る。 | 2等空佐, 1等空尉, 1等空曹, 3等空曹, 2等空士 |
| 目標（50～100文字） | 職種・階級に応じた具体的かつ測定可能な目標。 | 国際的なサイバー防衛演習の計画と実行を主導し、参加国との情報共有体制を確立する。（情報保証目標） |
| 重要度 | 目標の重要度を「〇」「△」「ー」で評価。 | 〇, △, ー |
| 困難度 | 目標達成の困難度を「〇」「△」「ー」で評価。 | 〇, △, ー |
| 自己申告（50～100文字） | 目標に対する達成度や自己評価を具体的に記述。 | 演習計画を策定・実施し、参加国との情報共有手順を確立。演習後の評価は非常に高かった。 |
| 上司コメント（50～100文字） | 自己申告に対する上長からの評価と次なる指導を記述。 | 複雑な国際連携を成功させた。確立した体制を恒常的な情報保証活動に組み込むこと。 |
| 評価 | 最終的な人事評価を4段階で記述。 | 卓越して優秀, 非常に優秀, 優秀, 良好 |

## 4. 生成するデータの基本情報
* **リストのテーマ**: 航空自衛隊 階級・職種別 目標・課題設定リスト
* **対象階級（均等に混合させる）**: 2等空佐、1等空尉、1等空曹、3等空曹、2等空士
* **職種（均等に混合させる）**: 操縦、要撃管制、整備、警備、通信、情報、輸送、会計、需品、施設、衛生など
* **目標範囲**: **201件目から400件目まで**の**200件**のデータ

## 5. 出力開始の指示
上記のルールを厳守し、「階級」から始まるヘッダー行を含めたCSV形式のデータを、201件目から400件目まで（合計200行）生成してください。
